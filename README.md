# lpphpadst126 - Clínica Estética
Sistema de gestão para Clínica Estética - LP-ADST1 2026

## Módulos
- Usuários (admin e funcionários)
- Clientes
- Produtos (estoque)
- Serviços (com produtos utilizados)
- Relatórios de produtos e serviços

## Instalação
1. Importar `DAL/clinica.sql` no phpMyAdmin
2. Acessar `http://localhost/lpphpadst126/`
3. Login: `admin` / senha: `123`
